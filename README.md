 > 基于vue全家桶的响应式音乐app，界面风格仿咪咕音乐

# 技术栈  
* vue全家桶：Vue.js,vue-router,vue-resource,vuex 
* axois 


# 功能展示  
## 音乐app
- 推荐页面
- 电台
- 热门歌单
- 推荐歌单
- 排行榜
- 歌手
- 播放页面
# 运行
* 安装：npm install
* 加载项目：npm run dev
* 监听8080端口
